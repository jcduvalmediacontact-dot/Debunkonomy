Debunk’Onomy — theme.css global unifié

Le fichier assets/theme.css a été créé et relié EN DERNIER dans toutes les pages HTML.
Il devient donc la couche de référence pour l'uniformisation visuelle.

Palette dominante :
- vert profond       --color-deep
- vert profond moyen --color-dark
- vert moyen         --color-mid
- bleu-vert          --color-teal
- canard             --color-duck
- gris foncé         --color-charcoal / --color-slate
- gris clair         --color-grey / --color-light
- blanc froid        --color-white

Pour modifier la charte, commence par modifier les variables :root dans assets/theme.css.

Les anciens CSS spécifiques restent présents pour les mises en page locales.
Une passe légère a remplacé les principales anciennes couleurs sable/doré/beige par les variables du thème.
Le fichier THEME-UNIFICATION-REPORT.json résume les modifications.
